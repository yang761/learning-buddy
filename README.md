# 🌸 學習小夥伴 Pro 🌸
一個超可愛的學習倒計時器，從 00:00:00 開始自定義倒數時間，支援跨設備保存紀錄～

## ✨ 功能特點
- 🕒 從 0 開始，可手動加 1秒/10秒/1分/10分/1小時
- 🐰 隨機切換可愛動物表情 + 鼓勵語
- 💾 自動保存學習紀錄，換設備也不丟
- 📱 手機/電腦/平板都能完美適配
- 🎨 溫柔漸變背景，治愈系設計
